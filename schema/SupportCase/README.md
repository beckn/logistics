# SupportCase

A domain-agnostic customer support ticket used to record an issue raised between parties — its category, priority, lifecycle status, supporting attachments, and support interaction sessions. It is independent of any particular domain; domain-specific values are carried via the category descriptor and the extensible attribute bags. A SupportCase has an `UNRESOLVED` / `RESOLVED` lifecycle and underpins disputes: a Dispute cannot be raised until at least one unresolved SupportCase is attached.
